# takto_preference_optimization

Token-Level Adaptive KTO (TAKTO): Extending Prospect-Theoretic Alignment to Fine-Grained Preferences. Combines KTO unpaired data approach with token-level importance weighting and adaptive loss aversion.

## Project Structure

```
takto_preference_optimization/
├── context/           # Extracted paper contexts and profiles
├── ideas/             # Generated and approved research ideas
├── experiments/       # Experiment runs with configs and results
├── papers/
│   ├── drafts/        # Paper drafts in progress
│   └── final/         # Final paper versions
├── data/              # Datasets
├── figures/           # Generated figures
└── project.json       # Project configuration
```
