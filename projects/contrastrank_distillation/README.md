# contrastrank_distillation

Contrastive Reasoning Distillation for Efficient Neural Rerankers - transferring LLM reasoning capabilities to small models for document ranking

## Project Structure

```
contrastrank_distillation/
├── context/           # Extracted paper contexts and profiles
├── ideas/             # Generated and approved research ideas
├── experiments/       # Experiment runs with configs and results
├── papers/
│   ├── drafts/        # Paper drafts in progress
│   └── final/         # Final paper versions
├── data/              # Datasets
├── figures/           # Generated figures
└── project.json       # Project configuration
```
